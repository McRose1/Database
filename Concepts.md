# Concepts

## 语言类型
- DML (Data Manipulation) 数据操纵语言 - 作用在数据表中的元祖上
  - Insert
  - update
  - delete
- DDL (Data Definition) 数据定义语言 - 作用在数据表上
  - create
  - alter 
  - drop
  - rename
  - truncate 
- TCL (Transaction Control) 事务控制语句
  - commit 
  - rollback
  - savepoint
- DCL (Data Control) 数据控制语言
  - grant 
  - revoke
